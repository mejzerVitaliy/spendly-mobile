# Spendly Mobile

Personal finance tracker with AI-powered transaction input. Built with Expo (React Native).

---

## Features

### Onboarding

- **Product slides** — animated showcase of key app capabilities before sign-up
- **Guest mode** — full access without an account; data stored locally
- **Setup wizard** — select favorite categories, main currency, and set wallet opening balance
- **Interactive coach tour** — step-by-step tooltip guide to the main tab bar actions shown to new users

### Authentication

- Email/password registration and login
- Forgot password and reset password flows
- Guest → registered account upgrade
- Account deletion

### Transactions

**Manual creation:**
- Expense, income, or transfer between wallets
- Fields: amount, date, wallet, currency, category, description
- Recurring schedule: daily, weekly, every 2 weeks, monthly, every 6 months, yearly

**AI text input:**
- Type a natural-language description (e.g. "spent 38 lei on groceries")
- AI parses and returns a preview of one or more transactions
- Confirm all at once or tap individual rows to open a pre-filled edit form

**AI voice input:**
- Record audio describing the transaction
- AI transcribes and parses into a transaction preview
- Same confirmation flow as text input

**Transaction management:**
- Edit any transaction
- Duplicate a transaction (creates a copy dated today)
- Delete with confirmation dialog
- Long-press context menu with edit / duplicate / delete actions
- Haptic feedback on long-press

### Home Screen

- Income / expense balance summary for the selected period
- Filterable transaction list with debounced search
- Period selector (week / month / year) with prev/next navigation
- Pull-to-refresh for wallets, transactions and reports
- Sticky header that appears after scrolling past the balance section

### Analytics

- **Cash flow trend** — line/bar chart of income and expenses over time
- **Income vs expense ratio** — visual breakdown of the split
- **Category breakdown** — chart switchable between expense and income
- **Analytics / Insights tab** toggle

### AI Insights

- **Rule-based insights** — automatic cards for: cash flow status, savings rate, top expense category, spending concentration, peak spending day, active tracking days
- **AI Coach** — on-demand deep analysis generated by AI with personalized recommendations
- Usage counter and monthly limit indicator

### Wallets

- Multiple wallets with custom names and types
- Total balance hero card showing aggregate across all wallets in the main currency
- Set default wallet
- Archive and restore wallets
- Long-press action sheet: edit, set as default, archive
- Pull-to-refresh

### Settings

| Section | Details |
|---|---|
| Account | Edit email, change password, log out, delete account |
| Categories | Toggle favorite categories (up to 10) per type (expense / income); search |
| Currencies | Toggle favorite currencies (up to 5); change main currency; search |
| Language | English / Russian |
| Notifications | Enable / disable push notifications |
| Limits | View monthly AI usage (transactions + insights) with progress bars |
| Support & About | Privacy policy and Terms of Use links |

### Notifications

- Push notification opt-in with permission handling
- In-app notification list screen
- Notification types: daily check-in reminder, weekly financial summary, monthly recap, spending streak milestone

### Localization

- English and Russian — full UI coverage including validation messages, date formats, and number formatting

### Technical

- Offline detection with action guard — mutations blocked when offline with user-facing feedback
- Dark theme throughout with glass-morphism surface treatment on iOS
- Animated transitions: spring-based segment controls, sticky header fade, chart progress bars
- Haptic feedback on destructive or notable interactions

---

## Stack

- **Expo** (SDK 52) + **Expo Router** — file-based navigation
- **React Native** with **NativeWind** — Tailwind CSS styling
- **TanStack Query v5** — server state, caching, background refetch
- **Zustand** — client state with AsyncStorage persistence
- **React Hook Form** + **Zod** — forms and validation
- **React Native Reanimated** — animations
- **@gorhom/bottom-sheet** — modals and action sheets
- **expo-av** — audio recording for voice input
- **expo-blur** — glass effect on iOS
- **expo-haptics** — haptic feedback
- **i18next** — internationalization
